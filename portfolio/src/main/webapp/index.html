Projects

Here is a list of some of my past computer science projects:

  • Final Exam Scheduler
    • Implemented a final exam scheduler using vector of vectors of strings (representing the exam schedule) and a graph (representing all the courses) in Java.
    • The program reads in a list of Students and their classes, and uses a greedy algorithm to create a school-wide class schedule with the condition that any given student should not be double booked in the same exam block
  • Text Imitator
    • Implemented a vector (arraylist) of associations (similar to hashmaps) in Java to create a program that imitates the writing style of an inputted text.
    • Processed the input text by going through the text and counting and storing the frequency of each letter appearing after a certain letter or set of letters in a vector of associations. It then uses the data from the vector of associations to randomly generate letters according to what follows beforehand.
  • Ghostscript
    • Leveraged stacks in Java to create a partial implementation of Ghostscript, an interpreter for Postscript, a stack-based programming language used for generating vector graphics
    • Verified that Postscript could be successfully interpreted through test-driven development
